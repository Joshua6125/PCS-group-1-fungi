## Managing Packages (uv)

### Add a new library
To add a library (e.g., scipy):
```bash
uv add scipy
```

### Remove a library
```bash
uv remove numpy
```

### Update all packages
```bash
uv sync --upgrade
```

### Activate venv
```bash
source .venv/bin/activate
```
